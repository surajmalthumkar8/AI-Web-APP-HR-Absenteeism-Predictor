# HR Absenteeism Predictor

AI-powered HR Decision Support System for predicting employee absenteeism.

## Features
- **ML Predictions**: XGBoost model with SHAP explanations
- **LLM Explanations**: Natural language insights via Hugging Face Inference API
- **NLP Queries**: Ask questions in plain English
- **Analytics Dashboard**: Trends, distributions, and insights

## Tech Stack
- Backend: FastAPI + XGBoost + SHAP
- Frontend: React + TypeScript + TailwindCSS
- LLM: Hugging Face Inference API (Mistral-7B)

## Usage
1. Navigate to **Dashboard** for overview
2. Use **Predictions** to get AI-powered absence forecasts
3. Ask questions in **Ask AI** using natural language

Built as a portfolio demonstration of enterprise AI application development.
